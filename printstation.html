One-Time Print Station Setup
  1. 1
    In Windows, open Settings → Bluetooth & devices → Printers & scanners. Click ZDesigner ZP 450-200dpi and set it as the default printer.
  2. 2
    Find the Google Chrome shortcut on the desktop (or create one). Right-click it → Properties.
    In the Target box, add --kiosk-printing after the closing quote:
    "C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing
  3. 3
    Close this Chrome window completely, then re-open Chrome using that modified shortcut. Navigate back to this page (printstation.html).
  4. 4
    Click Confirm & Activate below. Then use Test Print to verify a label prints on the ZP450 with no dialog box appearing — if a dialog appears, the shortcut wasn't saved correctly.
This confirmation is saved on this computer. You won't see this again unless you click Reconfigure.
